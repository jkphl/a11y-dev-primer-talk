#### Tastaturkürzel

* `Page Up`, `←`, `↑`: Previous slide
* `Space`, `Return`, `Page Down`, `→`, `↓`: Next slide
* `c`: Toggle mouse cursor
* `i`: Toggle interactive mode (stepwise display of list items)
* `Esc`: Slide overview
